## Installation

CRUDE-Futures is NOT YET available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CRUDE-Futures', :git => 'https://github.com/JasonCanCode/CRUDE-Futures.git'
```
